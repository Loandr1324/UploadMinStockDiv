1. Программа "Подготоква файла для загрузки МО.exe" должна находится в той же папке, что и файл "Анализ установленного МО.xlsx".
2. После того как заказчик проверил все данный в файле "Анализ установленного МО.xlsx", он его сохраняет и запускает программу "Подготовка файла для загрузки МО.exe".
3. В результате выполнения программы, создаётся файл "Загрузка МО по подразделению.xlsx"
4. В обработке Минимальные остатки, с ранее установленным отобором по папкам или сегменту, выбираем какие МО подразделения надо установить и нажимаем "Перейти к МО".
5. Далее нажимаем кнопку Загрузить... и выбираем созданный файл "Загрузка МО по подразделению.xlsx".
6. После загрузки просматриваем, что всё загрузилось корректно и нажимаем "Применить".
